# 现代汉语词典离线版 Windows 应用程序开发计划

## 1. 项目概述

开发一个可在 Windows 系统上离线使用的现代汉语词典桌面应用程序，提供词汇查询、拼音显示、释义展示等功能，无需网络连接即可使用。

## 2. 技术选型

| 层面 | 技术 | 理由 |
|------|------|------|
| 编程语言 | Python 3.x | 生态丰富，开发效率高 |
| GUI 框架 | PyQt6 | 成熟的跨平台桌面框架，界面美观 |
| 数据库 | SQLite | 轻量级嵌入式数据库，适合离线场景 |
| 词典数据 | CC-CEDICT（开源） | 免费开源的中英文词典数据 |
| 打包工具 | PyInstaller (One-File 模式) | 将 Python 应用、解释器、所有依赖打包为单个 exe 文件 |

### 关于无需 Python 环境的说明

最终交付的应用程序是一个独立的 `.exe` 可执行文件，用户电脑上无需安装任何 Python 环境。PyInstaller 会将以下全部内容打包到 exe 文件中：
- Python 解释器本身
- PyQt6 GUI 库
- SQLite 数据库驱动
- 所有第三方依赖库
- 应用程序代码和资源文件

用户只需双击 exe 文件即可运行，体验与原生 Windows 应用完全一致。

### 备选方案（如需更小的体积）

如果 PyInstaller 打包体积过大，可考虑以下备选方案：
- **Nuitka**: 将 Python 编译为 C++ 再编译为原生 exe，体积更小，启动更快
- **PyQt6 + cx_Freeze**: 另一种打包方案

## 4. 功能清单

### 4.1 核心功能
- 按中文词汇搜索
- 显示拼音、词性、释义
- 支持模糊搜索
- 搜索历史记录

### 4.2 界面功能
- 搜索框实时匹配建议
- 词汇详情展示区
- 收藏词汇功能
- 深色/浅色主题切换

### 4.3 数据管理
- 首次运行时自动导入词典数据
- 支持更新词典数据
- 本地缓存最近查询

## 6. 数据库设计

### 表：words
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增 ID |
| traditional | TEXT | 繁体词 |
| simplified | TEXT | 简体词 |
| pinyin | TEXT | 拼音 |
| definition | TEXT | 英文释义 |

### 表：favorites
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增 ID |
| word_id | INTEGER | 关联 words.id |
| created_at | DATETIME | 收藏时间 |

### 表：search_history
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增 ID |
| word | TEXT | 搜索词 |
| searched_at | DATETIME | 搜索时间 |
